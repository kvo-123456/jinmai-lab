# 三个主题优化计划

## 现状分析
- 系统目前有三个主题：`light`、`dark`、`pink`
- `dark`主题已有较完整的样式定义
- `light`主题只有基础样式，需要增强
- `pink`主题几乎没有样式定义，需要完整实现
- 主题切换顺序：light → dark → pink → light

## 优化目标
- 完善三个主题的样式定义
- 确保主题间的一致性和完整性
- 增强视觉效果和交互体验
- 优化主题切换的动画效果

## 优化内容

### 1. pink 主题实现
- 完整的粉色系色彩方案
- 卡片、按钮、输入框等组件样式
- 排版和文字样式
- 交互反馈效果
- 滚动条样式
- 表单元素样式
- 动画和过渡效果

### 2. light 主题优化
- 增强卡片和按钮样式
- 优化输入框和表单元素
- 改进排版和文字层次
- 增强交互反馈效果
- 优化滚动条样式
- 完善动画效果

### 3. dark 主题优化
- 完善现有样式细节
- 增强发光效果和阴影
- 优化动画和过渡效果
- 改进排版细节
- 增强交互反馈
- 完善表单元素样式

## 技术实现
1. 在 `index.css` 中添加完整的 `.pink` 主题样式
2. 优化 `.light` 主题的现有样式
3. 增强 `.dark` 主题的视觉效果
4. 确保三个主题使用一致的 CSS 变量结构
5. 优化主题切换的过渡动画

## 预期效果
- 三个主题都有完整的样式定义
- 主题切换流畅自然
- 每个主题都有独特的视觉风格
- 增强用户交互体验
- 保持品牌一致性
