# 实现点击切换功能，隐藏品牌选择框展现粒子效果

## 需求分析
用户希望添加一个点击切换功能，能够隐藏品牌选择卡片，让粒子效果更完整地展示出来，提升视觉体验。

## 实现方案

### 1. 添加状态管理
- 在 `ParticleArt` 组件中添加 `showBrandCards` 状态，默认为 `true`
- 用于控制品牌选择卡片的显示和隐藏

### 2. 添加切换按钮
- 在顶部标题区添加一个切换按钮，用于控制品牌卡片的显示/隐藏
- 按钮位置：在标题下方，与描述文本对齐
- 按钮样式：采用渐变背景，与整体UI风格保持一致
- 按钮文字：根据状态动态显示"隐藏品牌卡片"或"显示品牌卡片"
- 按钮图标：使用Font Awesome的眼睛/眼睛斜杠图标

### 3. 动画效果实现
- 使用 `framer-motion` 实现平滑的过渡动画
- 显示/隐藏时添加淡入淡出和缩放效果
- 按钮自身添加悬停和点击反馈

### 4. 修改品牌选择区
- 为品牌选择卡片容器添加条件渲染
- 当 `showBrandCards` 为 `false` 时，隐藏品牌卡片区域
- 保留标题区域，让用户知道可以切换回来

### 5. 优化粒子效果展示
- 当品牌卡片隐藏时，粒子效果将占据更多视觉空间
- 确保粒子系统能够自适应布局变化

## 代码修改点

1. **添加状态**：在 `ParticleArt` 组件中添加 `showBrandCards` 状态
2. **添加切换按钮**：在标题区添加切换按钮
3. **修改品牌选择区**：为品牌卡片容器添加条件渲染
4. **添加动画效果**：使用 `framer-motion` 实现平滑过渡

## 预期效果
- 初始状态：显示品牌选择卡片，用户可以选择不同品牌
- 点击切换按钮：品牌选择卡片平滑隐藏，粒子效果完全展现
- 再次点击切换按钮：品牌选择卡片平滑显示，恢复初始状态
- 整个切换过程流畅自然，提升用户体验

## 技术实现
- 使用 React Hooks 管理状态
- 使用 framer-motion 实现动画效果
- 使用 Tailwind CSS 实现样式
- 保持与现有代码风格一致
